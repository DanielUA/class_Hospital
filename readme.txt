1. У каждого пациента есть итория болезни.
2. В истории болезни сохраняется 10 последних болезней.
3. Истрия болезни имеет информацию: когда пациент заболел, кто его лечил, когда выздровел,
процедуры и лекарства которые назначил врач.
4. Врач может назначить процедуры и лекарства. У рача и пациента есть имя и фамилия. 
5. У пациента есть номер страховки, У врача есть должность.
6. Пациент не может менять историю болезни
